# backstage-vue3

一款基于Vue3，JSON配置化UI组件库，同时兼容element-ui、ant-design-vue

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```

### Lints and fixes files

```
npm run lint
```
